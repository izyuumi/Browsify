# Browsify

A macOS menu bar application that intelligently routes HTTP/HTTPS URLs to different browsers or desktop applications based on user-defined rules.

## Features

- Route URLs to specific browsers based on domain patterns
- Support for browser profiles (Chrome, Firefox, Arc, etc.)
- Desktop app integration (Zoom, Teams, Slack, etc.)
- Interactive browser picker when no rule matches
- Automatic tracking parameter removal
- Menu bar interface
